# 🚀 GitHub 仓库和 PR 创建指南

## 方式1：手动安装 GitHub CLI（推荐）

### 1. 安装 Homebrew（如果没有安装）
在终端中运行：
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### 2. 安装 GitHub CLI
```bash
brew install gh
```

### 3. 登录 GitHub
```bash
gh auth login
```
选择：
- GitHub.com
- HTTPS
- 使用您的 GitHub 凭据登录

### 4. 创建 GitHub 仓库并推送代码
```bash
cd "/Users/lucas/claude code"

# 创建公开仓库
gh repo create claude-code-builder --public --description "🚀 AI-powered code generation platform - Lovable Clone with Claude API integration"

# 添加远程仓库
git remote add origin https://github.com/$(gh api user --jq .login)/claude-code-builder.git

# 推送代码
git push -u origin main
git push -u origin feature/connect4-game
```

### 5. 创建 Pull Request

gh pr create --title "feat: Add Connect 4 game demo and comprehensive documentation" --body "

## 方式2：通过 GitHub 网页界面

如果 GitHub CLI 安装困难，可以使用网页界面：

### 1. 创建 GitHub 仓库
1. 访问 [github.com/new](https://github.com/new)
2. 仓库名称：`claude-code-builder`
3. 描述：`🚀 AI-powered code generation platform - Lovable Clone with Claude API integration`
4. 选择 **Public**
5. 点击 **Create repository**

### 2. 推送代码到 GitHub
```bash
cd "/Users/lucas/claude code"

# 添加远程仓库（替换 YOUR_USERNAME 为您的 GitHub 用户名）
git remote add origin https://github.com/YOUR_USERNAME/claude-code-builder.git

# 推送代码
git push -u origin main
git push -u origin feature/connect4-game
```
